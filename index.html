Step 1 — Add the Apps Script
  1. Open the sheet above → click Extensions → Apps Script
  2. Delete everything in the editor

Step 2 — Deploy as Web App

  2. Type: Web app

  4. Who has access: Anyone
  5. Click Deploy → copy the Web App URL
